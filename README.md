# Micro-N - Microblogging & Chat API 

**Currently in active development.**

Being build with
[Nest framework](https://github.com/kamilmysliwiec/nest), TypeORM (MySQL), Redis, Nodemailer, JWT, class-validator and Jest.

## Installation & start

```
$ npm install
$ npm run start
```

## Tests

```
$ npm run test:watch
$ npm run e2e
```
